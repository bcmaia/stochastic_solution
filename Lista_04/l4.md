## Questão 10
Partı́culas radioativas são emitidas de uma fonte de acordo com um processo de
Poisson de intensidade $\lambda = 1$ partı́cula por minuto.

### Tópico A
Suponha que cinco partı́culas são emitidas no primeiro minuto. 
Qual é a probabilidade que exatamente duas dessas partı́culas foram emitidas nos
primeiros 30 segundos? (Resp.: 0.312).

___

Usando minutos como nossa unidade de tempo, o que queremos calcular pode ser expresso por:
$$ Pr \{ N(0.5) = 2 | N(1) = 5\} $$

Aplicando a formula de probabilidade condicional:
$$ Pr \{ A | B \} = \frac {Pr \{ A, B \}} {Pr \{ B \}} $$
$$ 
    Pr \{ N(0.5) = 2 | N(1) = 5\}
        = \frac {
            Pr \{ N(0.5) = 2, N(1) = 5\}
        } {
            Pr \{ N(1) = 5 \}
        }
$$

Para tornar os eventos independentes, vale:
$$ 
    Pr \{ N(0.5) = 2 | N(1) = 5\}
        = \frac {
            Pr \{ N(0.5) = 2, N(1 - 0.5) = 5 - 2\}
        } {
            Pr \{ N(1) = 5 \}
        }
$$

Dessa forma,
$$ 
    Pr \{ N(0.5) = 2 | N(1) = 5\}
        = \frac {
            { Pr \{ N(0.5) = 2 \} \cdot Pr\{ N(1 - 0.5) = 5 - 2 \} }
        } {
            Pr \{ N(1) = 5 \}
        }
$$

$$ 
    Pr \{ N(0.5) = 2 | N(1) = 5\}
        = \frac {
            { Pr \{ N(0.5) = 2 \} \cdot Pr\{ N(0.5) = 3 \} }
        } {
            Pr \{ N(1) = 5 \}
        }
$$

Aplicando a formula do Processo de Poisson:
$$ 
    Pr \{ N(t) = n\} 
        = \frac {
            e^{-\lambda t} (\lambda t)^n 
        } {
            n!
        }
$$

Para $n = 2$ :
$$
    Pr\{ N(0.5) = 2 \}
        = \frac {
            e^{-1 \cdot \frac 1 2} (1 \cdot \frac 1 2)^2 
        } {
            2!
        }
        = \frac { e^{-0.5} (0.5)^2 } { 2 } 
        = \frac {\frac 1 4 \sqrt{\frac 1 e}} {2}
        = \frac {\sqrt{\frac 1 e}} {8}
        \approx 0.07581633246407918
$$

Para $n = 3$ :
$$
    Pr\{ N(0.5) = 3 \}
        = \frac {
            e^{-1 \cdot \frac 1 2} (1 \cdot \frac 1 2)^3 
        } {
            3!
        }
        = \frac { \frac 1 8 \sqrt{\frac 1 e}  } { 6 } 
        = \frac { \sqrt{\frac 1 e} } { 48 } 
        \approx 0.012636055410679864
$$

Para calcular $Pr \{ N(1) = 5 \}$ :
$$
    Pr\{ N(1) = 5 \}
        = \frac {
            e^{-1 \cdot 1} (1 \cdot 1)^5
        } {
            5!
        }
        = \frac { e^{-1} \cdot 1^{5} } { 5! } 
        = \frac { 1 } { 120 e } 
        \approx 0.0030656620097620196
$$

Então
$$
    Pr \{ N(0.5) = 2 | N(1) = 5\}
        = \frac {
            \frac {\sqrt{\frac 1 e}} {8} \cdot \frac { \sqrt{\frac 1 e} } { 48 } 
        } {
            \frac { 1 } { 120 e } 
        }
        = \frac {\sqrt{\frac 1 e} \cdot \sqrt{\frac 1 e}} {8 \cdot 48 } 
            \cdot \frac { 120 e } { 1 } 
        = \frac {{\cancel e^{-1}} \cdot 120 \cancel e} {8 \cdot 48}
$$
$$
    Pr \{ N(0.5) = 2 | N(1) = 5\}
        = \frac {120} {8 * 48}
        = \frac {120} {384}
        = \frac {5} {16}
$$

### Tópico B
Seja $S_n$ o tempo de emissão da n−ésima partı́cula. 
Expresse $S_n$ em termos do tempo de permanência no estando $n$,
isto é $T_n$ , e então encontre $E[S_n]$. 
(Resp.: E[Sn ] = n)

___

O tempo de emissão da n−ésima partı́cula ($S_n$ ) depende do tempo de
permanência no estando $T_n$ e do tempo de permanencia nos estados anteriores,
ou seja:
$$ S_n = T_1 + T_2 + T_3 + \dots + T_{n-1} + T_n $$
$$ S_n = \sum _{i = 1} ^{n} T_i $$

Em uma distribuição de poisson, o tempo de emissão da n-ésima partícula segue
uma distribuição exponêncial.
$$ T_n \sim Exp(\lambda) $$

Em uma distribuição exponencial $Exp(\lambda)$, podemos calcular a 
média esperada (a esperânça) por:
$$ E[T_n] = \frac 1 \lambda $$

Dado $\lambda = 1$, tem-se que:

$$ E[T_n] = 1 $$

Sabemos que se trata de um 
<strong> processo de poisson homogênio </strong>,
então podemos inferir que:
$$ E[T_i] = 1  \hspace{1em}  \forall \, i$$

Assim, obtemos:
$$ E[S_n]
    = E \left[
        \sum _{i = 1} ^{n} T_i  
    \right] 
$$
$$ E[S_n]
    = E \left[
        \sum _{i = 1} ^{n} 1 
    \right] 
$$

Usando a propriedade de soma de constantes:
$$ \sum _{i = 1} ^{n} C = C \cdot n $$

Temos então:
$$ E[S_n] = 1 \cdot n $$

Portanto, concluímos que:
> . 
> $$ E[S_n] = n $$
> .
